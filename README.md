# Spring security + JWT 테스트 토이프로젝트
Spring boot 4.0.x, JAVA2.1<br/>

## 기능
1. refresh token을 활용한 멀티로그인, 강제로그아웃, 토큰 재갱신
3. spring security 커스텀 활용
4. 간단 API 구현
5. Custom APIExcpetion
6. Controller Exception 처리

## API
회원가입. 로그인. 로그아웃.
